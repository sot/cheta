cheta.update_client_archive#

Synchronize local client copy of cheta archive to the server version.

This uses the bundles of sync data that are available on the ICXC server to update the local HDF5 files that comprise the cheta telemetry archive. It also updates the archfiles.db3 that serve as a date index for queries.

Exceptions#

Exception for row mismatch between existing archive and available sync update
